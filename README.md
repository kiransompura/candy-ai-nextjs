# Next.js

## Prerequisites

- Node.js (v18 or later recommended)
- npm (comes with Node.js)

Check versions:

```bash
node -v
npm -v
```

## Installation

Clone the repository:

```bash
git clone https://github.com/kiransompura/candy-ai-nextjs.git
cd path-where-project-cloned
```

Install dependencies:

```bash
npm install
```

## Development

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Production Build

Build the project:

```bash
npm run build
```

Start the production server:

```bash
npm start
```
